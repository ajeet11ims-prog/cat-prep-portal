# CAT Prep With Ajeet Sir - Final V3 Working Portal

This is a static Vercel-ready student test portal.

## Upload steps
1. In GitHub repo root, delete wrong duplicate files if present:
   - portal-calm-theme.css
   - profit-loss-tests-addon.js
   - index(1).html
   - app(1)(1).js
   - styles(1)(1).css
2. Upload/replace all files from this ZIP into the repo root.
3. Keep the existing `tests/` folder with all HTML test files.
4. Commit changes and wait for Vercel deployment.
5. Open `https://cat-prep-portal-26.vercel.app/?v=finalv3#home` and press Ctrl + F5.
