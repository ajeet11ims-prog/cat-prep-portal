# CAT Prep With Ajeet Sir - Static Portal

## Upload / replace root files

Upload these files to the GitHub repo root:

- `index.html`
- `styles.css`
- `app.js`
- `tests-data.js`
- `tests.json`
- `attempt.html`
- `attempt.js`
- `admin.html`
- `admin.css`
- `admin.js`
