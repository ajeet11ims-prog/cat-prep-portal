Manual GitHub delete process:
  1. Select the blank test from the dropdown.
  2. Click Copy HTML File Path.
  3. Click Remove Test from Index and download the updated tests-data.js.
  4. In GitHub, open the copied path and delete or replace that HTML file.
  5. Upload the updated tests-data.js in repo root and wait for Vercel.
